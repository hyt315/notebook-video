# Narration audio

Generated narration is intentionally not bundled with the open-source template.

Before rendering, place licensed narration at `audio/narration.mp3`, retain word timing at `audio/narration.mp3.json`, then run the project sync command. Record the TTS provider, voice, generation date and redistribution terms in your project notes.

The included timing JSON is structural test data for captions; it does not grant rights to any voice or audio recording.
